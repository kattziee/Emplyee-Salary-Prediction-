# Emplyee-Salary-Prediction-

Employee Salary Prediction using Machine Learning

**Project Overview**

This project is a Machine Learning-based web application that predicts an employee's salary based on factors such as education level, job title, age, gender, and years of experience.

The application uses a trained machine learning model and an interactive Streamlit interface to provide real-time salary predictions. It demonstrates the complete machine learning workflow, from data preprocessing and model training to deployment.

**Key Features**

Predict employee salaries in real time
User-friendly Streamlit web application
Automated data preprocessing and feature transformation
Machine Learning model training and evaluation
Clean and modular project structure
Deployment-ready application

**Project Workflow**

1. Data Collection

Employee salary data is collected and loaded into the project.

2. Data Preprocessing

Handle missing values
Encode categorical features
Scale numerical features

3. Model Training

A machine learning model is trained using employee-related attributes to learn salary patterns.

4. Salary Prediction

The trained model predicts salary based on user inputs.

5. Deployment

The application is deployed using Streamlit for easy access and real-time predictions.
